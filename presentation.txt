1. Задача курсовой работы:
  В курсовой работе была поставлена задача выполнить верстку одностроничного веб-сайта по предоставленному макету и загрузить ее на свой репозиторий в GitHub.

2. Ресурсы для реализации поставленной задачи:
Макет одностраничного сайта в Figma
Редактор кода Visual Studio Code 
Репозиторий GitHub

3.Этапы работы:
Просмотр макет. Вырезка из него всего необходимого, что может пригодиться: иконки, картинки и прочую графику. Также запись шрифтоф, которые используются в макете.
Вёрстка html и css. Сначала верстальщик пишет структуру в html. Заодно он сразу вешает элементам классы и идентификаторы, чтобы позднее стилизовать их в css. Нормальный верстальщик должен придумывать адекватные имена классов. Например, menu, head, widget и т.п. Чтобы было понятно, что это за элемент. Стилизация должна выполняться по порядку. Сначала нужно прописать какие-то общие правила. Например, общий фон странички, базовый шрифт. Последующие работы зависят от опыта и професионализма разработчика.

4. Результат работ: 
  Одностраничный сайт на репозитории в GitHub.

В результате прохождения курса по WEB-разработке мной были получены навыки 
  по работе с программой Visual Studio Code, и многими другими сайтами, созданными для упращения работ разработчиков, такими как Figma, GitHub, Tilda.
